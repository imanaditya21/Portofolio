Microcontroller Programming (Arduino)

Pelatihan & Praktik IoT

Deskripsi Kegiatan

Microcontroller Programming (Arduino) merupakan kegiatan pelatihan yang berfokus pada penggunaan Arduino sebagai mikrokontroler utama untuk mengontrol perangkat elektronik serta membangun sistem otomasi berbasis Internet of Things (IoT).

Pelatihan ini mencakup pemahaman dasar hingga lanjutan mengenai pemrograman Arduino, pengolahan input dari sensor, pengendalian output, serta integrasi perangkat keras dengan sistem berbasis jaringan.

Tujuan Pelatihan

  • Memahami konsep dasar mikrokontroler dan Arduino
  • Menguasai pemrograman Arduino menggunakan Arduino IDE
  • Mampu mengontrol perangkat elektronik secara otomatis
  • Mengimplementasikan sistem IoT sederhana hingga menengah

Materi yang Dipelajari

  • Pengenalan Arduino Uno dan board sejenis
  • Struktur program Arduino (setup & loop)
  • Pembacaan sensor (LDR, Ultrasonic, IR, dll)
  • Kontrol output (LED, relay, motor, buzzer)
  • Komunikasi serial dan debugging
  • Integrasi Arduino dengan modul IoT (NodeMCU / ESP8266)

Implementasi dan Praktik

Dalam pelatihan ini dilakukan berbagai praktik langsung, seperti mengendalikan lampu otomatis, sistem monitoring jarak, serta otomasi perangkat menggunakan sensor. Peserta juga mempelajari cara menghubungkan Arduino dengan jaringan internet untuk monitoring dan kontrol jarak jauh.

Hasil dan Manfaat

  • Mampu membuat sistem kontrol perangkat berbasis Arduino
  • Memahami alur kerja sistem embedded dan IoT
  • Meningkatkan keterampilan problem solving hardware & software
  • Siap mengembangkan proyek IoT secara mandiri
